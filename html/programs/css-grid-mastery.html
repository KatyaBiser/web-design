CSS Grid Mastery

6 weeks • Intermediate • Led by Prof. Marcus Grid

Program Overview

Stop clearing your floats. Stop wrapping everything in flexbox containers. It's time to admit the truth: you've been avoiding CSS Grid because it seemed scary. Twenty-three nested divs with "display: flex" is NOT a layout strategy—it's a cry for help.

This intensive 6-week program will guide you through the five stages of Grid grief: denial ("floats still work fine"), anger ("why are there so many track sizing options?!"), bargaining ("maybe I can just use flexbox..."), depression (staring at grid-template-areas at 2 AM), and finally, acceptance (your first perfectly aligned gallery that works on mobile).

Led by Prof. Marcus Grid, who once built an entire e-commerce site using only CSS Grid and sheer willpower. No frameworks. No JavaScript for layout. Just raw, unapologetic Grid.

What You'll Learn

  • Understanding the difference between grid-template-columns and grid-column (yes, they're different)
  • Creating responsive layouts without a single media query (your mind will be blown)
  • Using fr units like you mean it—no more guessing percentages
  • Grid-template-areas: naming your grid like you're naming your pets
  • Auto-fit vs auto-fill: the existential crisis every Grid developer faces
  • Implicit vs explicit grids (when Grid decides to help you... whether you asked or not)
  • Subgrids: grids within grids (we need to go deeper)
  • Grid + Flexbox: yes, you can use both (plot twist!)
  • Debugging grid layouts without throwing your laptop out the window

Prerequisites

This program is for developers who:

  • Have been using float: left since 2010 and refuse to let go
  • Think "clearfix" is a permanent CSS solution
  • Wrap everything in flexbox containers "just in case"
  • Have written "display: inline-block; vertical-align: top" more times than they'd like to admit
  • Are tired of calculating percentages for column widths
  • Have 47 Stack Overflow tabs open about centering things
  • Are ready to commit 8-10 hours per week to Grid enlightenment
